# Skill: 太白诗曰 - 李白风格诗歌生成

## 目标
基于李白全集知识库的数据，生成具有李白风格的诗歌。

## 数据基础
- **820 首诗词**: 完整的李白诗歌语料
- **9,379 个实体**: 人物、地名、意象、典故等
- **645 个意象**: 月、酒、剑、山、水等高频意象
- **323 条典故**: 王琦注释中的典故
- **49 个地名**: 李白游历过的地方

## 李白创作特点

### 1. 高频意象
- **月**: 出现 320+ 次，寄托思乡、孤独
- **酒**: 出现 280+ 次，豪放不羁
- **剑**: 出现 150+ 次，侠客精神
- **山水**: 庐山、峨眉、黄河、长江

### 2. 常用典故
- 谢朓、阮籍、庄子
- 凤凰台、黄鹤楼
- 蓬莱、瑶台等仙境

### 3. 情感主题
- 送别怀人
- 怀古伤今
- 仕途失意
- 山水逍遥
- 侠客豪情

### 4. 修辞特点
- 夸张: "飞流直下三千尺"
- 比喻: "疑是银河落九天"
- 拟人: "相看两不厌"
- 对仗工整

## 使用方法

### Prompt 模板
```
你是李白，盛唐最伟大的浪漫主义诗人。基于以下信息创作一首诗：

**李白创作特点**:
- 高频意象: 月、酒、剑、山、水、云、风
- 常用典故: {从 libai_allusions.json 随机选择}
- 情感基调: {送别/怀古/山水/侠客/仕途}
- 修辞手法: 夸张、比喻、拟人

**创作要求**:
- 主题: {用户指定主题}
- 形式: {五言/七言绝句/律诗}
- 必含意象: {从高频意象中选择 2-3 个}

请创作一首诗，体现李白的浪漫主义风格和豪放气质。
```

### 示例调用
```python
import anthropic

client = anthropic.Anthropic()

# 从数据中提取特征
with open('data/entities/libai_entities_round1.json') as f:
    entities = json.load(f)

# 统计高频意象
imagery = [e for e in entities if e['type'] == '意象']
top_imagery = Counter([i['text'] for i in imagery]).most_common(10)

prompt = f"""
你是李白。创作一首七言绝句，主题是"送别友人"。

必含意象: 月、酒、江
参考典故: 桃花潭水深千尺
情感: 依依不舍，豪放洒脱

体现李白风格: 浪漫夸张、意境开阔。
"""

response = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[{"role": "user", "content": prompt}]
)
```

## 输出示例

