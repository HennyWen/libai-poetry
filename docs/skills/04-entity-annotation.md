# 实体标注规范（李白全集）

## 11 类实体定义

| 类型 | 说明 | 正例 | 反例 |
|------|------|------|------|
| 人物 | 真实历史人物（诗人、帝王、将相等） | 杜甫、汉武帝、孟浩然 | 嫦娥（神话人物） |
| 地名 | 地理位置（山川、城市、关隘等） | 峨眉山、长安、玉门关 | 故乡（泛指，无法定位） |
| 意象 | 跨诗传统的象征性符号 | 月、梅、雁、酒、剑 | 子规（标为自然物，除非明确作为意象使用） |
| 典故 | 有出处的历史/神话引用事件 | 鸿门宴、昭君出塞、烂柯人 | 杜甫（人名，不是典故） |
| 时令 | 时间指代（季节/时辰/年号/节日） | 清明、中秋、暮春、天宝年间 | 今日（无法定位的时间词） |
| 情感主题 | 内容创作检索标签 | 送别、边塞、怀古、山水 | 悲（单字情感词，不作主题） |
| 诗体/词牌 | 诗歌形式 | 七律、五绝、乐府、古风 | 不标注（从诗体字段获取） |
| 典籍 | 援引的古典文献 | 《诗经》、《庄子》、《史记》 | 杜甫诗（非典籍，是人物） |
| 神话仙道 | 神仙/神话人物和仙界地点 | 嫦娥、织女、瑶台、蓬莱 | 汉武帝（真实历史人物） |
| 自然物 | 具体动植物/自然现象 | 子规、芙蓉、飞瀑、春雷 | 月（高频意象，优先标为意象） |
| 官职 | 官职名称 | 翰林、御史、刺史、将军 | 臣（泛指，非具体官职） |

## 边界规则

1. **意象 vs 自然物**：同一词可能两者都有，优先判断诗中是否有象征功能：
   - "月" → 意象（思乡/孤独等象征含义）
   - "子规" → 自然物（具体鸟类，除非出现"子规啼月""子规声里"等明显意象化用法）

2. **地名取舍**：
   - "故乡""天涯""他乡" → 泛指，**不标注**
   - "长安""洛阳""峨眉山" → 可定位，**标注**

3. **人物 vs 典故**：
   - 诗中直接提及某人（"忆杜甫""与孟浩然"） → 人物
   - 引用某人做过的事（"效荆轲之壮志"） → 典故

## 输出格式（JSON）

```json
{
  "poem_id": "libai_0001",
  "entities": [
    {
      "text": "明月",
      "type": "意象",
      "canonical": "月",
      "note": "思乡意象"
    },
    {
      "text": "故乡",
      "type": "地名",
      "canonical": null,
      "note": "泛指，不标注经纬度"
    }
  ]
}
```

字段说明：
- `text`：诗中原文
- `type`：11类之一
- `canonical`：规范化名称（别名时填写，如"皓月"→"月"；相同时填 null）
- `note`：说明（可选，用于歧义或重要备注）
